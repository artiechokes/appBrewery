The Simon Game

Press A Key to Start

Press Any Key to begin the Game. Simon will give the first signal. Repeat the signal by pressing the same color.

Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same color, in order. Simon will duplicate these first two signals and add one.

Continue playing as long as you can repeat each sequence correctly.

If you fail to repeat a sequence exactly, Simon responds with a "GAME OVER" sound. This means you've lost, and the sequence of signals ends.
